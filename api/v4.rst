Logout
~~~~~~

.. http:post:: /api4/Users/logout

  Logout from device

  **Parse as json**:

  .. sourcecode:: json

    {
      "code": "123456"
    }

  **Response as json (correct code)**:

  .. sourcecode:: json

    {
      "logout": true
    }

  **Response as json (incorrect code)**:

  .. sourcecode:: json

    {
      "logout": false
    }

Delete

POST /api4/Reports/delete_hf_activity_case

Delete case

Query Parameters

  • year (int) – year of report, e.g: 2020,2021,…

  • month (string) – month of report, e.g: 01, 02, …, 12

  • hc_code (string) – code of health facility

  • rec_id (int) – primary key of row
